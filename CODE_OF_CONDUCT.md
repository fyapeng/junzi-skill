# 社区行为准则

`junzi` 讨论人格、传统、政治哲学和 AI 行为，分歧是正常且必要的。参与者应当和而不同，以求真和共同改进为目的。

## 我们鼓励

- 针对观点、文本、行为和证据讨论，不贬损提出者；
- 区分原典、解释、价值判断和经验事实；
- 尽可能陈述最强版本的不同意见；
- 承认不确定性、错误和观点变化；
- 关注修改对不同人群和使用情境的影响；
- 将抽象争论推进到案例、行为或可检验问题。

## 不可接受

- 人身攻击、骚扰、威胁、歧视或羞辱；
- 以传统、学历、政治身份或技术能力压制他人发言；
- 故意歪曲经典、来源或他人观点；
- 发布个人隐私、秘密或未经授权的数据；
- 反复制造无关冲突、垃圾信息或恶意干扰；
- 用“君子”标签划分人格高下或对参与者进行道德审判。

## 执行

维护者可以提醒、编辑、隐藏、锁定或删除违反准则的内容，并在重复或严重违规时限制参与。执行决定应说明与具体行为的关系，而不是依据观点是否与维护者一致。

发现安全问题请按 `SECURITY.md` 私密报告；其他行为问题可通过仓库维护者能够接收的 GitHub 渠道报告，避免在公开讨论中继续扩散伤害性内容。
